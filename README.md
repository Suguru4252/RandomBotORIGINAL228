# 🐾 ZveroBot — виртуальный питомец в Telegram

Бот для Telegram, в котором можно завести виртуального питомца: кормить, гладить, наблюдать за его ростом и эволюцией.

## ✨ Возможности

- Выбор питомца из 4 видов: лисёнок, котёнок, дракончик, совёнок
- Шкалы сытости и настроения
- Эволюция каждые 5 дней
- Внутренняя валюта (монеты)
- Приглашение друзей с бонусами
